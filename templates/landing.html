Frequently Asked Questions

MindSpace bundles a CSV dataset file `dataset/student_burnout.csv`. On system startup, the data cleaning pipeline in `utils.py` reads it using Pandas, filters duplicate rows, imputes missing values using medians, and automatically saves cleaned entries into SQLite database tables.

We utilize a Scikit-Learn `RandomForestClassifier` trained on indicators like sleep hours, study hours, screen time, attendance, physical activity, and stress levels. It yields ~90% prediction accuracy with feature importances available to read.

Admin users have role-based authorization to modify the database. They can perform CRUD operations on student records, download raw files, trigger ML model retraining, and view system audit logs.
